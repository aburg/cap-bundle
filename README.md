# Cap Bundle

Protect your symfony projects with a free and open source catcha solution.

This will work out of the box if:

- you can provide a running instance of [Cap](https://trycap.dev/)
- you are using Symfony Flex
- you are using Symfony UX with Stimulus and Turbo
- you are running AssetMapper

## Installation

```[bash]
composer require aburg/cap-bundle
```
